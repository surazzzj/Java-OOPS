# 🚀 Object-Oriented Programming (OOP) in Java

This repository contains Java programs demonstrating the core concepts of Object-Oriented Programming (OOP). Each file focuses on a specific OOP principle with simple and easy-to-understand examples.

## 📚 Topics Covered

### 1. Encapsulation
- `Encapsulation.java`
- `EncapsulationWithConstructor.java`

Encapsulation is the process of binding data and methods into a single unit (class) and restricting direct access using private variables.

### 2. Abstraction
- `Abstraction.java`
- `Abstraction2.java`

Abstraction hides implementation details and shows only essential functionality to the user.

### 3. Inheritance
- `InheritanceWithOverriding.java`
- `MultipleInheritance.java`

Inheritance allows one class to acquire properties and methods of another class, promoting code reusability.

### 4. Polymorphism
#### Method Overloading
- `MethodOverloading.java`
- `ConstructorOverloading.java`

Allows multiple methods or constructors with the same name but different parameters.

#### Method Overriding
- `MethodOverriding.java`
- `InheritanceWithOverriding.java`

Allows a subclass to provide a specific implementation of a method already defined in its parent class.

### 5. Interfaces
- `Interface.java`

Interfaces provide complete abstraction and support multiple inheritance in Java.

### 6. Constructors
- `Constructor.java`
- `ConstructorOverloading.java`

Constructors are special methods used to initialize objects.

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/object-oriented-programming.git
```

2. Navigate to the project folder

```bash
cd object-oriented-programming
```

3. Compile the Java file

```bash
javac FileName.java
```

4. Run the program

```bash
java FileName
```

Example:

```bash
javac Encapsulation.java
java Encapsulation
```
